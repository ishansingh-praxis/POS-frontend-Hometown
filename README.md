# HomeTown Point POS - Basic React Version

Converted into a normal Vite + React frontend with simple client-side routing.

## Removed

- framework-specific router/start tooling
- generated route tree
- SSR server/start files
- project-cloud metadata

## Used now

- React
- Vite
- React Router DOM
- Existing UI components and POS screens
- LocalStorage demo authentication

## Run

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```
